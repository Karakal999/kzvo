# Інформація про проєкт

## Академія Педагогічної Освіти

### Технологічний стек

#### Основні технології
- **React 18** - Сучасна бібліотека для побудови UI
- **TypeScript** - Строга типізація для надійного коду
- **Vite** - Швидкий збірник нового покоління
- **Tailwind CSS 3** - Утилітарний CSS-фреймворк
- **React Router DOM 7** - Клієнтська маршрутизація
- **Lucide React** - Красиві SVG іконки

### Кольорова палітра

Проєкт використовує фірмові кольори Академії:

#### Основний синій (Primary)
- **HEX**: `#1e3a8a`
- **Використання**: Основний колір бренду, шапка, підвал, кнопки
- **Tailwind класи**: `bg-primary`, `text-primary`, `border-primary`

#### Акцентний жовтий (Accent)
- **HEX**: `#fbbf24`
- **Використання**: Акценти, іконки, ховер-ефекти, заклики до дії
- **Tailwind класи**: `bg-accent`, `text-accent`, `border-accent`

### Основні компоненти

#### Header (Шапка)
- Адаптивна навігація з мобільним меню
- Логотип з іконкою академії
- Навігаційні посилання на всі сторінки
- Плавні переходи та ховер-ефекти

#### Footer (Підвал)
- Контактна інформація
- Швидкі посилання
- Інформація про академію
- Копірайт

#### MainLayout
- Обгортка для всіх сторінок
- Header + контент + Footer
- Flexbox для притискання підвалу до низу

#### Home (Головна сторінка)
- Hero-секція із закликом до дії
- Секція переваг (4 картки)
- Call-to-action секція
- Використання іконок Lucide

### Сторінки

1. **Головна (/)** - Представлення академії, основні переваги
2. **Про нас (/about)** - Інформація про академію, місія
3. **Програми (/programs)** - Освітні програми
4. **Курси (/courses)** - Каталог доступних курсів
5. **Новини (/news)** - Останні новини та події
6. **Контакти (/contacts)** - Контактна форма та інформація

### Команди розробки

```bash
# Встановлення залежностей
npm install

# Запуск у режимі розробки (http://localhost:5173)
npm run dev

# Збірка для production
npm run build

# Попередній перегляд production-збірки
npm run preview

# Перевірка коду (ESLint)
npm run lint
```

### Особливості реалізації

1. **Адаптивний дизайн** - Працює на всіх пристроях (mobile-first підхід)
2. **TypeScript** - Повна типізація для надійності
3. **Tailwind CSS** - Швидка розробка з утилітарними класами
4. **React Router** - SPA з клієнтською маршрутизацією
5. **Lucide Icons** - Сучасні, легковагі іконки
6. **Модульна структура** - Легко масштабована архітектура
