# Академія Педагогічної Освіти

Сучасний веб-сайт для Академії педагогічної освіти, побудований з використанням актуальних технологій.

## Технологічний стек

- **React 18** - Бібліотека для побудови користувацького інтерфейсу
- **TypeScript** - Типізований JavaScript для підвищення надійності коду
- **Vite** - Швидкий збірник для сучасних веб-додатків
- **Tailwind CSS** - Утилітарний CSS-фреймворк для швидкої розробки UI
- **React Router DOM** - Маршрутизація для односторінкового додатку
- **Lucide React** - Сучасні SVG іконки

## Структура проєкту

```
src/
├── components/     # Переиспользовані компоненти (Header, Footer)
├── layouts/       # Макети сторінок (MainLayout)
├── pages/         # Сторінки сайту (Home, About, Programs, тощо)
├── hooks/         # Кастомні React хуки
├── utils/         # Допоміжні функції
├── types/         # TypeScript типи та інтерфейси
├── data/          # Мокові дані для розробки
└── assets/        # Статичні файли (зображення, шрифти)
```

## Кольорова схема

Проєкт використовує фірмові кольори Академії:

- **Основний синій**: `#1e3a8a` (primary)
- **Акцентний жовтий**: `#fbbf24` (accent)

Кольори налаштовані в `tailwind.config.js` і доступні через класи Tailwind:
- `bg-primary`, `text-primary` - основний синій
- `bg-accent`, `text-accent` - акцентний жовтий

## Встановлення та запуск

### Встановлення залежностей

```bash
npm install
```

### Запуск у режимі розробки

```bash
npm run dev
```

Додаток буде доступний за адресою `http://localhost:5173`

### Збірка для продакшену

```bash
npm run build
```

### Попередній перегляд продакшен-збірки

```bash
npm run preview
```

## Розробка

### Додавання нових компонентів

Створюйте нові компоненти в папці `src/components/`:

```typescript
// src/components/MyComponent.tsx
const MyComponent = () => {
  return <div>My Component</div>;
};

export default MyComponent;
```

### Додавання нових сторінок

1. Створіть компонент сторінки в `src/pages/`
2. Додайте роут в `src/App.tsx`

### Робота з типами

Визначайте TypeScript типи в `src/types/index.ts`:

```typescript
export interface MyType {
  id: string;
  name: string;
}
```
