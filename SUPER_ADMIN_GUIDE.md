# 👑 Інструкція для Головного Адміністратора

## Що змінилось

**admin@kzvo.edu** тепер є головним суперадміністратором з повними правами доступу до всього сайту.

---

## ✨ Візуальні зміни

### 1. На сторінці профілю (`/profile`)

При вході як admin@kzvo.edu, ви побачите:

```
┌─────────────────────────────────────────┐
│  👤  Головний Адміністратор  [SUPER ADMIN] │
│      admin@kzvo.edu                      │
│      🛡️ Адміністратор                    │
└─────────────────────────────────────────┘
```

- **Золотий бейдж "SUPER ADMIN"** біля вашого імені
- Роль відображається як "Адміністратор"

### 2. В адмін-панелі (`/admin`)

В списку користувачів:

```
┌─────────────────┬──────────────────┬─────────────────┬──────────────┐
│ Ім'я            │ Email            │ Роль            │ Дії         │
├─────────────────┼──────────────────┼─────────────────┼──────────────┤
│ Головний Адмін  │ admin@kzvo.edu   │ [SUPER ADMIN] ✨│ Захищений   │
│ Іван Петров     │ ivan@kzvo.edu    │ Адміністратор   │ [Змінити]   │
│ Марія Сидорова  │ maria@kzvo.edu   │ Вчитель         │ [Змінити]   │
└─────────────────┴──────────────────┴─────────────────┴──────────────┘
```

- **Золотий градієнт** для вашого акаунту
- Мітка "Захищений акаунт" - нікому не можна вас видалити
- Інші адміни не можуть змінити вашу роль

## 🔐 Ваші права доступу

### ✅ Повний доступ:

#### Користувачі:
- Перегляд всіх користувачів
- Створення нових користувачів
- Зміна ролей (admin, teacher, student)
- Видалення користувачів
- Призначення інших адміністраторів

#### Контент:
- Створення, редагування, видалення новин
- Управління подіями
- Завантаження документів
- Модерація заявок

#### Система:
- Доступ до всіх налаштувань
- Зміна критичних параметрів
- Перегляд статистики
- Управління вчителями та програмами

---

## 🛡️ Захист вашого акаунту

### Що захищено:

1. **Не можна видалити**
   - Навіть інші адміни не можуть видалити admin@kzvo.edu
   - Firestore Rules блокують видалення

2. **Не можна змінити роль**
   - Ваша роль "admin" захищена
   - Тільки ви можете змінювати ролі інших

3. **Повні права всюди**
   - Firestore Rules дають вам доступ до всього
   - Немає обмежень на читання/запис

## 📋 Що ви можете робити

### 1. Управління користувачами

**Створити нового адміністратора:**
1. `/admin` → "Користувачі"
2. Знайдіть користувача в списку
3. Натисніть "Зробити адміном"

**Зняти адміністратора:**
1. `/admin` → "Користувачі"
2. Знайдіть адміна в списку
3. Натисніть "Прибрати адміна"

### 2. Управління контентом

- **Новини:** Додавання, редагування, видалення
- **Події:** Створення та управління
- **Документи:** Завантаження та організація

### 3. Модерація

- **Заявки студентів:** Схвалення/відхилення
- **Реєстрації на події:** Перегляд та управління

## ⚠️ Важливі правила безпеки

### 1. Захист паролю

```
✅ ПРАВИЛЬНО:
- Пароль мінімум 12 символів
- Комбінація букв, цифр, символів
- Унікальний пароль (не використовується інде)

❌ НЕПРАВИЛЬНО:
- admin123
- password
- kzvo2025
```

### 2. Двофакторна автентифікація

Увімкніть 2FA в Firebase Console:
1. Authentication → Settings
2. Enable Email Verification
3. Enable 2FA for admin accounts

### 3. Регулярний перегляд

Щотижня перевіряйте:
- Список адміністраторів (`/admin`)
- Історію активності (`/profile` → "Активність")
- Нові користувачі

---

## 🚨 Що робити, якщо...

### Забули пароль

1. На сторінці входу: "Забули пароль?"
2. Введіть `admin@kzvo.edu`
3. Перевірте email
4. Створіть новий безпечний пароль

### Підозрілва активність

1. Перейдіть в `/profile` → "Активність"
2. Перевірте список дій
3. Якщо щось підозріле → змініть пароль
4. Перевірте список адміністраторів

### Потрібна допомога

Технічна документація:
- `FIREBASE_QUICK_SETUP.md` - налаштування Firebase
- `setup-super-admin.md` - детальна інструкція
- `firestore.rules` - правила безпеки
